## 👨‍💻 Sobre Mim

Sou um desenvolvedor focado em construir soluções eficientes, unindo design moderno e automações inteligentes. Atualmente dedico minha rotina ao estudo de **Arquitetura de Software**, **Desenvolvimento Web/Mobile** e **Integrações de APIs**. 

Tenho facilidade para resolver problemas complexos sob pressão e busco minha primeira oportunidade oficial como **Desenvolvedor Júnior**. Acredito que a combinação entre design limpo e automação de processos é a chave para transformar linhas de código em valor real de mercado.

- 🎓 Cursando **Ensino Superior / Técnico na área de Tecnologia**
- 📍 Baseado em **Maringá, PR, Brasil**
- 🚀 Fundador/Desenvolvedor na **XSITES** — Soluções Web de Alta Performance
- ⚡ Comprometido com a **constância diária** e a excelência técnica
